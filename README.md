## ğŸ¨ Customization

### Theme Colors
The platform uses CSS custom properties for easy theming:

```css
:root {
  --lm-pink: #ff62b0;      /* Primary accent */
  --lm-purple: #6b46c1;    /* Secondary accent */
  --lm-dark: #1f1d29;      /* Dark background */
  --lm-darker: #0a0014;    /* Darker background */
}
```

### Modifying the Theme
1. Open any HTML file
2. Find the `<style>` section
3. Modify the CSS variables in `:root`
4. Save and refresh

### Custom LM Character
Replace `LM mask.png` with your own character image (recommended size: 200x200px)
